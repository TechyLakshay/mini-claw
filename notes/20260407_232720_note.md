# Java `for` Loop Example

Below is a simple, well‑structured Java program that demonstrates how to use a `for` loop to iterate over a range of numbers and print each value.

```java
public class ForLoopDemo {
    public static void main(String[] args) {
        // Loop from 1 to 10 inclusive
        for (int i = 1; i <= 10; i++) {
            System.out.println("Iteration " + i);
        }
    }
}
```

## How It Works

- **Initialization**: `int i = 1` – sets the loop counter to 1.
- **Condition**: `i <= 10` – continues looping while `i` is less than or equal to 10.
- **Increment**: `i++` – increases `i` by 1 after each iteration.

## Running the Code

1. Save the file as `ForLoopDemo.java`.
2. Compile: `javac ForLoopDemo.java`
3. Run: `java ForLoopDemo`

You should see output similar to:

```
Iteration 1
Iteration 2
...
Iteration 10
```

Feel free to modify the range or the body of the loop to suit your needs!
